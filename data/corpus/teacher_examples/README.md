# 教师优秀计划书分类放置规范

建议按类别建子文件夹，每个类别放对应计划书：

```txt
data/corpus/teacher_examples/
  环境保护/
  科技创新/
  医疗健康/
  教育服务/
  乡村振兴/
  智能制造/
  文旅文创/
  社会治理/
  其他/
```

## 命名建议

文件名建议：`年份_学校_项目名_结果.pdf`  
示例：`2025_XX大学_智能垃圾分类系统_省赛金奖.pdf`

## 备注

- 允许放 `pdf/docx/pptx/txt/md`。
- 子文件夹名称会被后端识别为 `category`。
- 后续图谱抽取会优先读取这些分类信息。
