# 管理员端功能总览

## 1. 全局大盘
- 展示全校项目总数、平均分、风险触发次数、注册用户数、证据总数、系统健康度等关键指标。
- 支持项目类别分布、团队一览（可跳转用户列表）。

## 2. 用户与权限管理
- **用户列表**：可按角色（学生/教师/管理员）、团队、关键词筛选。
- **用户操作**：添加、批量创建、批量导入（支持 CSV/Excel）、修改角色、禁用/启用、重置密码、删除。
- **团队管理**：支持团队重命名、成员增删、教师团队管理。
- **批量创建**：支持学生/教师账号一键生成，教师可分配多个团队及邀请码。
- **统计**：显示各类用户数量。

## 3. 教师表现大盘
- 展示教师人数、覆盖学生数、平均分、风险率、干预覆盖率等。
- 支持多维度排序（综合排名、平均分、风险率、活跃学生、干预覆盖率）。
- 明细表：每位教师的团队数、学生数、活跃学生、平均分、风险率、干预覆盖率、最近活跃时间。

## 4. 教学干预监控
- 汇总全校干预任务总数、参与教师数、被干预学生数、已完成干预数。
- 教师干预聚合表：每位教师的干预任务分状态统计。
- 最近干预记录：时间、教师、学生、标题、状态、作用范围、项目ID。

## 5. 项目总览
- 所有班级、学生提交的项目一览，含项目ID、学生、团队、评分、风险规则、提交时间。
- 支持项目数量、平均分、涉及团队数等统计。

## 6. 高频漏洞看板
- 展示全校范围内被触发最多的业务漏洞 Top 排行，含规则名称、触发次数、影响比例、严重等级。
- 高风险项目清单：项目编号、名称、类别、风险数。

## 7. 访问与安全日志
- 记录所有用户的关键操作与越权访问尝试。
- 统计总请求数、成功/错误/拦截次数、平均耗时、热门 API 路径。
- 日志明细：时间、用户、操作类型、请求路径、详情、状态。

## 8. 系统健康度详情
- 综合评分，分为项目质量、风险控制、系统稳定性、教学干预四大维度。
- 各维度详细指标与权重一览。
